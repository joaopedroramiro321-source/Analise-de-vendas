# Análise de Vendas

## Objetivo
Analisar dados de vendas para gerar insights estratégicos.

## Ferramentas
- Python
- Pandas
- Matplotlib

## Análises realizadas
- Faturamento total
- Faturamento por categoria
- Produtos mais vendidos
- Produtos mais lucrativos

## Principais insights
A categoria dos eletrônicos foi a mais bem sucedida, tendo o notebook como o top produto da
categoria e das vendas como um todo, contudo o produto que teve o maior volume de vendas foram os mouses,
provavelmente por conta de seu baixo preço.

Mesmo que o mouse tenha sido o produto mais vendido, não foi nem de perto o produto que gerou maior
faturamento, sendo superado por produtos com maior preço, nesse quesito. Esses dados sugerem que, para
aumentar a efiencia e lucratividade da empresa, é indicado alocação de maiores esforços nas vendas de
produtos mais caros, ou premiuns.
